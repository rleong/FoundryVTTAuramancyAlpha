  • The Cost for learning a new Path is equal to: 2 + (2 x the number of other Paths you have) EXP
  • The Cost for advancing an existing Path is equal to: 2 + (the current Tier of the Path) EXP


  • Cost: 2 + (1 x the number of times you’ve used this suboption for a particular Perk Category) EXP
    • Perk Categories: Combat Perks, Major Arcane Perks.
  • Cost: 1 + (1 x the number of times you’ve used this suboption for a particular Perk Category) EXP
    • Perk Categories: Non-Combat Perks, Crafting Perks, Operating Perks, Minor Arcane Perks.


  • Cost: 1 + (1 x the number of times you’ve used this suboption for a particular Proficiency Subject) EXP
    • Gain a Proficiency in a new Proficiency Subject.
  • Cost: 2 + (1 x the number of times you’ve used this suboption for a particular Proficiency) EXP
    • Gain Expertise in an existing Proficiency.
